### File Structure

There are 4 types of Duck: IndianHomeDuck, AmericanFlyingDuck, RubberDuck, WoodenDuck

Duck.java (superclass)   
    |--> IndianHomeDuck.java  
    |--> AmericanFlyingDuck.java  
    |--> RubberDuck.java  
    |--> WoodenDuck.java  
    
QuackBehavior.java (Interface)  
    |--> Quack.java  
    |--> Squeak.java  
    |--> MuteQuack.java  
    
FlyBehavior.java (Interface)  
    |--> FlyNoWay.java   
    |--> FlyWithWings.java
